# TOMATO

A complete **MERN stack food ordering application** that allows users to browse food items, add them to cart, place orders, and make secure online payments using **Stripe**.

---

## 📌 Overview

This project is a real-world **food delivery web application** built using **React JS, Node.js, Express, MongoDB**, and **Stripe Payment Gateway**.  
It includes a user-facing frontend, an admin panel, and a backend server to handle authentication, orders, and payments.

---

## 🚀 Features

- User authentication (Signup / Login)
- Browse food items
- Add & remove items from cart
- Secure online payments using Stripe
- Place food orders
- Order status tracking
- Admin panel to manage food items and orders

---

## 🛠 Tech Stack

- **Frontend:** React JS  
- **Backend:** Node JS, Express JS  
- **Database:** MongoDB  
- **Payment Gateway:** Stripe  
- **Authentication:** JWT  

---

## 🧩 Project Structure

- Frontend (User Interface)
- Admin Dashboard
- Backend REST API

---

## 💳 Payment Integration

- Integrated Stripe for secure online payments  
- Orders are placed only after successful payment  

---

## ▶️ How to Run the Project

1. Clone the repository  
2. Install dependencies for frontend and backend  
3. Configure environment variables (MongoDB URI, Stripe keys, JWT secret)  
4. Start the backend server  
5. Start the frontend application  

---

## 🎯 Purpose

This project is built for learning and practicing **full stack MERN development**, backend APIs, authentication, and payment gateway integration.
